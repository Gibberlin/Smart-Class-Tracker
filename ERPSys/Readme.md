## 🚀 Getting Started

### **Prerequisites**
* 🐘 **XAMPP/WAMP** installed on your local machine.
* 🌐 **Web Browser** (Chrome, Firefox, or Edge).
* 💡 **Basic Knowledge** of PHP/MySQL for configuration.

### **Setup Instructions**

1.  **Clone the Repo**
    ```bash
    git clone https://github.com/Dexter0013/Student-Management-System.git
    ```

2.  **Import Database**
    * Open **phpMyAdmin** (`http://localhost/phpmyadmin`).
    * Create a new database named `student_db`.
    * Click **Import** and select the `database.sql` file located in the `/sql` folder.

3.  **Connect & Run**
    * Place the project folder in `C:/xampp/htdocs/`.
    * Open `config.php` and ensure your credentials match (default is usually `root` with no password).
    * Launch your browser and visit: `http://localhost/Student-Management-System`.
